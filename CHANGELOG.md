# Changelog

## v1.1.0

为服务器状态插件新增可配置的网络使用情况与流量显示项，并将状态文本生成改为异步执行。

新特性：

- 引入网络使用情况和网络总流量状态项，以便进行更细致的网络监控。
- 允许详细状态命令可选地显示新增加的网络相关信息。
- CPU占用项添加了几核几线程的信息。

增强改进：

- 重构状态项获取器的注册方式，改为使用绑定的可调用对象而不是方法名字符串。
- 在后台线程中生成简要和详细状态文本，以避免阻塞事件循环。
- 改进运行时长和数据大小的格式化，包括本地化的运行时长字符串和更紧凑的单位标签。

文档：

- 更新 README，记录新的网络状态项，并澄清可配置的状态显示选项。
